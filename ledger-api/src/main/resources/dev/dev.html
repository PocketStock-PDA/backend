5) 실시간 호가 (WebSocket / STOMP · UH1) 미연결
종목코드

6) 해외 실시간 호가 (KIS · HDFSASP0) 티커
상단 5번에서 먼저 STOMP 연결. 순수 티커(AAPL)만 보내면 서버가 거래소·세션(정규/주간)을 자동판별해 tr_key(DNAS…/RBAQ…)를 조립한다(#33) — 더이상 접두사 직접 안 붙임. 토픽 키도 안정적 티커(SYMB). 미국 정규장(한국 밤)·주간거래(10~16시) 시간에 데이터 수신.
7) 해외 실시간 체결가 (KIS · HDFSCNT0) 티커
상단 5번 연결 후 /topic/foreign/transaction/{티커} 구독 → 해외 실시간 체결가 push. 순수 티커(AAPL)로 구독(서버가 세션 자동판별 #33, 접두사 직접 안 붙임).
8) 국내 실시간 체결가 (LS · US3) 종목코드
상단 5번 연결 후 /topic/stock/trade/{code} 구독 → 국내 실시간 체결가(통합 체결, 현재가·등락·체결량) push.
9) 실시간 환율 USD/KRW (LS · CUR)
상단 5번 연결 후 /topic/currency/usd-krw 구독 → LS 실시간(CUR) 현물USD 환율 push. 종목코드 없는 단일 토픽. 외환시장 개장시간(평일 주간)에 데이터 수신.
