3) 국내 호가 (t8450) 종목코드

4) 온주 주문 (whole) + 보유/내역 예수금 미조회

5) 실시간 호가 (WebSocket / STOMP · UH1) 미연결
종목코드
STOMP /ws 연결 후 /topic/asking/{code} 구독 → LS 실시간(UH1) push 수신. 구독 시작 시 LS 등록, 해제 시 LS 해제(온디맨드).
6) 해외 실시간 호가 (KIS · HDFSASP0) KIS코드
상단 5번에서 먼저 STOMP 연결. KIS tr_key = R/D + 시장(3) + 종목. 야간/정규 DNAS·DNYS·DAMS, 미국주간(10~16시) RBAQ·RBAY·RBAA (예: DNASAAPL, RBAQAAPL). 미국 정규장(한국 밤) 또는 주간거래 시간에 데이터 수신.
